🎯 اپلیکیشن شما به صورت PWA ساخته شده!

این یک Progressive Web App است که می‌تواند روی هر دستگاهی (موبایل، تبلت، دسکتاپ) به صورت اپلیکیشن واقعی نصب شود.

📱 اندروید 🍎 iOS 💻 ویندوز 🌐 وب

1انتشار روی سایت vanizan.com

ساده‌ترین راه انتشار، آپلود فایل‌ها روی سایت خودتانه:

  • از طریق هاست cPanel، فایل‌های پوشه dist رو آپلود کنید
  • یا با FTP به سرور vanizan.com متصل بشید
  • یا از Git + Vercel/Netlify استفاده کنید
💡 بعد از آپلود، کاربران می‌تونن آدرس vanizan.com رو باز کنن و روی "افزودن به صفحه اصلی" بزنن تا اپ نصب بشه!

2انتشار در گوگل پلی (Google Play)

الف) تبدیل PWA به APK با PWABuilder

  1. به سایت pwabuilder.com برید
  2. آدرس https://vanizan.com رو وارد کنید
  3. گزینه Package for Android رو انتخاب کنید
  4. فایل APK آماده رو دانلود کنید

ب) یا با Bubblewrap (روش حرفه‌ای)

npm install -g @aspect-dev/aspect-cli
npm install -g @bubblewrap/cli
bubblewrap init --manifest=https://vanizan.com/manifest.json
bubblewrap build

ج) ثبت در گوگل پلی

  1. حساب Google Play Developer بسازید (25$ یکبار)
  2. به play.google.com/console برید
  3. روی Create App کلیک کنید
  4. اطلاعات اپلیکیشن رو وارد کنید:
    • نام: گرداب ثروت ونیزان
    • توضیح کوتاه: باورهای قدرتمند ثروتمند شدن در کوتاه‌مدت
    • دسته‌بندی: Lifestyle / Education
    • زبان: فارسی
  5. اسکرین‌شات‌ها و آیکون 512x512 آپلود کنید
  6. سیاست حریم خصوصی بنویسید
  7. APK/AAB رو آپلود کنید
  8. برای بررسی ارسال کنید (معمولاً 2-7 روز)

3انتشار در کافه بازار ☕

  1. به developer.cafebazaar.ir برید
  2. حساب توسعه‌دهنده بسازید
  3. APK ساخته شده رو آپلود کنید
  4. توضیحات، اسکرین‌شات و آیکون آپلود کنید
  5. محتوای فارسی و اسلامی به ارزش اپ اضافه کنید
  6. برای بررسی ارسال کنید (معمولاً 1-3 روز)
💡 کافه بازار برای اپ‌های فارسی و محتوای مذهبی/انگیزشی خیلی پرطرفدار است!
