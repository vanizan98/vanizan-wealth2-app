⚡ سریع‌ترین راه (پیشنهاد ما)

اپلیکیشن شما یک PWA کامل است. ساده‌ترین راه ساخت APK، استفاده از سرویس آنلاین PWABuilder مایکروسافت است که فقط چند دقیقه طول می‌کشد و نیازی به نصب هیچ نرم‌افزاری ندارد.

🚀 رفتن به PWABuilder.com
روش ۱ — آسان (بدون کدنویسی)

🌐 ساخت APK با PWABuilder

مناسب برای همه — فقط مرورگر لازم است

۱

آپلود سایت روی vanizan.com

ابتدا فایل‌های پوشه dist را روی هاست خود با HTTPS آپلود کنید. (HTTPS برای PWA الزامی است)

۲

ورود به PWABuilder

به سایت pwabuilder.com بروید و آدرس https://vanizan.com را وارد کنید و روی Start بزنید.

۳

بررسی امتیاز PWA

سایت به‌صورت خودکار manifest و service worker را بررسی می‌کند. باید امتیاز بالایی بگیرید (اپ شما همه موارد را دارد ✅).

۴

دانلود پکیج اندروید

روی Package For Stores → بخش Android → دکمه Generate Package کلیک کنید.

💡 دو گزینه دارید: «Test Package» برای نصب مستقیم (APK) و «Production» برای گوگل پلی (AAB).
۵

نصب روی گوشی

فایل .apk را در گوشی اندروید باز کنید و نصب کنید. (ممکن است نیاز باشد «نصب از منابع ناشناس» را فعال کنید)

روش ۲ — متوسط (خط فرمان)

🫧 ساخت APK با Bubblewrap (رسمی گوگل)

ابزار رسمی گوگل برای ساخت TWA — کنترل کامل

۱

پیش‌نیازها

نصب Node.js و JDK 17

۲

نصب Bubblewrap

npm install -g @bubblewrap/cli
۳

راه‌اندازی پروژه

bubblewrap init --manifest https://vanizan.com/manifest.json

فایل twa-manifest.json آماده در پروژه را می‌توانید جایگزین کنید.

۴

ساخت APK

bubblewrap build

فایل‌های app-release-signed.apk و app-release-bundle.aab ساخته می‌شوند.

⚠️ برای TWA باید فایل assetlinks.json را در مسیر vanizan.com/.well-known/assetlinks.json قرار دهید تا نوار آدرس مرورگر مخفی شود. (Bubblewrap راهنمایی می‌کند)
روش ۳ — حرفه‌ای (Capacitor)

⚡ ساخت APK با Capacitor

برای دسترسی به امکانات بومی اندروید (نوتیفیکیشن، فایل، و...) — اپ کاملاً آفلاین

۱

نصب Capacitor

# در پوشه پروژه
npm install @capacitor/core @capacitor/cli
npm install @capacitor/android
npx cap init

فایل capacitor.config.json آماده در پروژه موجود است.

۲

ساخت نسخه وب + افزودن اندروید

npm run build
npx cap add android
npx cap sync
۳

باز کردن در Android Studio

npx cap open android

پروژه در Android Studio باز می‌شود.

۴

ساخت APK نهایی

در Android Studio: منوی Build → Build Bundle(s) / APK(s) → Build APK(s)

فایل APK در مسیر android/app/build/outputs/apk/ ساخته می‌شود.

💎 با Capacitor اپلیکیشن کاملاً آفلاین کار می‌کند و نیازی به سرور یا اینترنت ندارد!
